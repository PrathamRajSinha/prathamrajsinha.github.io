Email Spam Detection using ML model: Random Forest

About the Project

Project 2

Email Spam Detection comparing multiple ML models

About the Project

In this study, I compared multiple machine learning models, including Gradient Boost, Neural Networks, Logis2c Regression, Support Vector Machine, and Linear Discriminant Analysis, for email spam detection. My findings indicate that Gradient Boost achieved the highest accuracy, outperforming the other models, making it a promising choice for effective email spam detection.
